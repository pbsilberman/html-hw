Summary: Latitude vs. X

This website will exhibit the results of the WeatherPy project. In that project, we randomly generated approximately 500 cities and built visualizations showing various weather outcomes vs distance from the equator. The various images that were created as part of the project are displayed on this website, as well as the raw data that was used in the analysis.
